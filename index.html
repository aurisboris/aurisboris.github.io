BAOS: Barangay Automated Office System

BAOS

FEATURES

  • User Management

    Basic CRUD functions for users. Allows for the Admin to manage the list of barangay officials and create user accounts for each. User permissions can be managed to control which user is able to perform specific functions in the application.

  • Schedule Management

    Allows for the Barangay Secretary to perform CRUD functions when setting up a schedule for the barangay officials. Schedules can be viewed in the website's dashboard and schedules can be set to either be viewable for all users or only to specific users.

  • Issue Barangay Certificates

    The Barangay Secretary has the ability to select a name from the list of barangay residents registered in the database and issue a barangay certificate based on the uploaded file template. The Secretary may create additional certificate types that they can issue and the Treasurer is able to set the amount to be collected for each certificate type.

  • Garbage Fee Collection

    Keeps a record of all outstanding balances for all households registereed in the database. The Treasurer is able to record payments from the residents representing the different households and is also able to set the amount for the fees and penalties upon late payment.

  • RCD Document Generation

    Generates a Report of Collections and Deposits Document based on the records of issued certificates, bank deposits, and garbage fee payments based on a specified date, year, or range of months.

  • Barangay Dispute Management

    Keeps a record of all filed complaints among the residents. Tracks the sessions conducted between the assigned lupon and the dispute parties as well as the status of each case.

  • Inventory Management

    Records all assets owned by the barangay officials. Allows for basic CRUD functions and computes for the total value of assets. Assets can be borrowed and reserved by barangay residents and the status of such would be tracked.

  • Community Management

    CRUD functions for all registered residents in the barangay. The list of residents can be viewed in its entirety or by household.
